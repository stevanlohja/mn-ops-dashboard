# Architecture Decision Records (ADRs)

MNF-owned ADRs (infrastructure, tooling, SRE process decisions) have moved to [`midnightntwrk/midnight-sre/docs/adr/`](https://github.com/midnightntwrk/midnight-sre/tree/main/docs/adr) so material that may name partner companies or expose operational specifics stays out of FNO-accessible spaces.

ADRs covering cross-org release coordination or FNO-facing operations may still land here in future. The local [template and guide](/docs/adr/architecture-decisions) remains as a starting point for any such ADRs.
